## 1. Objectif du TP

* effectuer des requêtes HTTP dans React
* utiliser les hooks `useState` et `useEffect`
* consommer une API via deux méthodes : **fetch()** (natif) et **axios**
* afficher l’état du chargement
* gérer les erreurs
* afficher les données reçues dans le composant
